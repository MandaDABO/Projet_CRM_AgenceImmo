🚀 CRM Immobilier Fullstack - Agence Nanterre
Ce projet est une solution de Gestion de la Relation Client (CRM) développée sur mesure pour une agence immobilière. Il permet de centraliser la gestion des prospects, de suivre la performance commerciale via des analyses de données et d'automatiser les campagnes de communication.

🛠️ Stack Technique
Framework : Next.js 15 (App Router, Turbopack)

Base de Données : Supabase (PostgreSQL)

Authentification : Supabase Auth

API Emailing : Brevo (ex-Sendinblue)

Styling : Tailwind CSS & Lucide Icons

Déploiement : Vercel

✨ Fonctionnalités Clés
📊 Dashboard Analytics
Calcul de la LTV (Life Time Value) : Identification des clients les plus rentables.

Répartition Géographique : Visualisation des leads par ville via des graphiques circulaires (PieCharts).

Suivi des Sources : Analyse de la provenance des leads (Marketing, Recommandations, etc.).

✉️ Centre de Communication
Templates Dynamiques : Utilisation de modèles d'emails avec variables de fusion (ex: {{prenom}}).

Programmation d'envois : Possibilité de planifier un email à une date et heure précises (Scheduling via Brevo API).

Historique des échanges : Traçabilité complète de toutes les communications envoyées par les agents.

👥 Gestion des Leads & Contacts
Segmentation : Distinction claire entre acheteurs et vendeurs.

Statuts de dossier : Suivi en temps réel de l'avancement (Nouveau, En cours, Gagné, Perdu).
